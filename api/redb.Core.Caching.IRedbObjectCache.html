Interface IRedbObjectCache

Namespace
redb.Core.Caching
Assembly
redb.Core.dll

Interface for caching WHOLE RedbObject objects (not just Props!) Cache key = objectId, Hash is used for validity validation Cache entire object with base fields + Props for:

  • 0 DB queries on Cache HIT (without SELECT base fields)
  • Reuse of nested RedbObject through references (memory savings)
