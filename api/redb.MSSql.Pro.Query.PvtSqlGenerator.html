PVT SQL generator for T-SQL syntax. Uses MAX(CASE WHEN) for scalar fields - single pass optimization. Uses STRING_AGG for array fields (SQL Server 2017+). Two-level CTE: raw_values (aggregation) → pvt_cte (filters). SECURITY: Parameterized SQL queries.

GeneratePvtCte(IReadOnlyList<FieldInfo>, string, string, string?)

Generates two-level CTE with MAX(CASE WHEN) optimization. Level 1 (raw_values): Aggregates values with MAX(CASE WHEN v._id_structure = X THEN v.column END) Level 2 (pvt_cte): Applies filters on aliased columns. Uses STRING_AGG for array fields.
