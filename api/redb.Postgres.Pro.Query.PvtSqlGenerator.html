GeneratePvtCte(IReadOnlyList<FieldInfo>, string, string, string?)

Generates CTE-based PVT (NOT correlated - single pass!). This is 47x faster than EXISTS-based approach on 15K objects.

Example output: WITH pvt_cte AS ( SELECT * FROM ( SELECT v._id_object, MAX(v._Long) FILTER (...) AS "Age", ... FROM _values v WHERE v._id_structure = ANY($1) AND v._id_object IN (SELECT _id FROM _objects WHERE _id_scheme = $2) GROUP BY v._id_object ) pvt WHERE pvt."Age" >= 30 )
