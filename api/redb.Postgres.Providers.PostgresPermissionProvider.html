PostgreSQL implementation of permission provider. Inherits all business logic from PermissionProviderBase. Only provides PostgreSQL-specific SQL dialect.

IMPORTANT: Uses PostgreSQL function get_user_permissions_for_object() for efficient recursive permission calculation.

Usage: services.AddScoped<IPermissionProvider, PostgresPermissionProvider>();
