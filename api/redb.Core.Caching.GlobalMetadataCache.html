Class GlobalMetadataCache

Namespace
redb.Core.Caching
Assembly
redb.Core.dll

Metadata cache with domain isolation. Combines scheme caching and CLR type registry (former AutomaticTypeRegistry). Instance is bound to specific domain, static data is shared. Each domain (typically one per database connection) has isolated cache.
