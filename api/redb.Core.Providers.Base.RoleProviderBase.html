Base implementation of role provider with common business logic. Database-specific providers inherit from this class and provide ISqlDialect.

Usage: public class PostgresRoleProvider : RoleProviderBase { public PostgresRoleProvider(IRedbContext context, IRedbSecurityContext security) : base(context, security, new PostgreSqlDialect()) { } }
