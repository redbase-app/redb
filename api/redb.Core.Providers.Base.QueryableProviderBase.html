Class QueryableProviderBase

Namespace
redb.Core.Providers.Base
Assembly
redb.Core.dll

Base class for IQueryableProvider implementations. Contains all common logic for creating LINQ queries. DB-specific implementations provide concrete query providers via factory methods.
