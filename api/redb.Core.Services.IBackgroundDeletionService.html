Interface IBackgroundDeletionService

Namespace
redb.Core.Services
Assembly
redb.Core.dll

Background deletion service with queue-based processing. Provides fire-and-forget deletion: mark objects, enqueue purge, return immediately. Purge runs in background thread with separate DB connection. Progress is stored in trash object in database (persistent).
