Class PermissionProviderBase

Namespace
redb.Core.Providers.Base
Assembly
redb.Core.dll

Base implementation of permission provider with common business logic. Database-specific providers inherit from this class and provide ISqlDialect.

IMPORTANT: Uses DB-specific SQL function for effective permissions! PostgreSQL: get_user_permissions_for_object() MSSQL: Must provide equivalent CTE/JOIN query
